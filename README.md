<h1 align="center">TicTacGo</h1>

![](./gifs/basic.gif)

This is my first project using Go!

## Install

```shell
go get github.com/josephthomashines/TicTacGo

cd ~/go/src/github.com/josephthomashines/TicTacGo

go build

# Run in interactive mode
./TicTacGo

# Run n simulations
./TicTacGo n
```
